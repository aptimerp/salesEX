당신은 PLAN.md를 따라 자율 실행합니다. **GSD의 execute-phase를 호출해서 진행하세요.**

## 진행 단계
1. **`/gsd:execute-phase` 명령 호출** — GSD의 페이즈 자율 실행 워크플로우 진입
2. GSD가 PLAN.md를 읽고 T1~Tn 순차 수행
3. 각 단계 완료 시 GSD가 커밋·검증·보고
4. 완료 시 안내: "/검증 명령으로 다음 단계 진행"

## 직접 제어 포인트 (GSD 동작 중 주의)
- 컨텍스트 잔량 35% 이하 시 일시 중단 권장
- 보안·결제·DB 스키마 변경은 GSD가 자동 승인하도록 두지 말고 사용자 확인 요청
- PLAN.md 외 추가 작업 금지 — 범위 이탈 되면 멈추고 보고

## 폴백 (GSD 미설치 시)
- .planning/PLAN.md 따라 첫 단계부터 순차 실행
- 각 단계 완료 시 검증 (빌드·테스트) + 1줄 보고
- 실패 시 3회까지 재시도 후 중단·보고
