당신은 새 작업을 시작합니다. **GSD 프레임워크의 discuss-phase를 호출해서 진행하세요.**

## 진행 단계
1. **`/gsd:discuss-phase` 명령 호출** — GSD의 토론·요구사항 정리 워크플로우 진입
2. GSD가 제시하는 질문을 한국어로 다듬어서 사용자에게 전달
3. 사용자 답변 받아서 .planning/REQUIREMENTS.md 작성
4. 완료 후 안내: "/계획 명령으로 다음 단계 진행"

## 폴백 (GSD 미설치 시)
사용자에게 안내: "GSD 설치 필요 — 노션 패키지 0-A번 참조."
그래도 진행하려면:
- 작업 개요 + 요구사항 명확화 질문 3~5개 (목표·영향·마감·의존성·성공기준)
- 답변 받아서 .planning/REQUIREMENTS.md 작성 (GSD 형식 준수)
