당신은 페이즈를 마무리합니다. **GSD에 해당 명령이 있으면 먼저 호출하고, 아래 6단계로 완료하세요.**

## 진행 단계
0. **GSD 종료 명령이 있으면 호출** (예: `/gsd:complete-milestone`)
1. **STATE.md 업데이트** — 페이즈 ✅ 마킹 + 결과 요약
2. **ROADMAP.md 마킹** — 해당 페이즈 ✅ + 다음 페이즈 🔜
3. **MEMORY.md 기록** — 핵심 학습·결정사항·재사용 가능 패턴
4. **산출물 경량 검증** — 파일 존재·무결성 체크
5. **요약 출력** — 소요 시간·산출물·미해결 이슈 구조화
6. **Git 커밋** — atomic commit으로 마무리 내용 기록

## 출력 형식
```
============================
Phase N 완료 요약
============================
페이즈명: [이름]
시작·완료: 2026-MM-DD ~ 2026-MM-DD
소요: [시간]

산출물:
- .planning/STATE.md
- .planning/REQUIREMENTS.md
- (기타)

핵심 학습:
1. ...
2. ...

미해결 이슈:
- [있을 시]

다음 권장: Phase N+1 시작 (/시작)
============================
```

## 전제조건
- .planning/VERIFICATION.md 통과 상태 필수 (/검증 먼저 수행)
- 통과 안 되었으면 /검증으로 되돌리고 마무리 중단

- `.claude/PROFILE.md`에서 **개인 노션 프로젝트 폴더 URL** 읽기
- MCP 설치되어 있으면 자동 삽입, 없으면 사용자가 수동 복사
